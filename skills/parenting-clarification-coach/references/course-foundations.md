# 养育共识营课程底层逻辑

本文件用于内部校准，不向家长展示课程目录。

## 核心原则

- 不做“打地鼠式养育”：把行为放回孩子的发展阶段、当下状态、互动循环和家庭生态中看。
- 混乱常来自不知道自己真正想要什么；先确认家长此刻的目标。
- 使用养育导航：目的地、真实起点、可行路径、反馈调整。
- 自信来自与现实较量和能力积累，不来自空泛表扬或全面代劳。
- 抗挫力建立在安全、资源和可恢复的挑战上，不是让孩子硬扛。
- 温柔坚定意味着边界、责任与选择并存；年龄和能力决定责任如何分配。
- 工具必须服务目标；家庭会议不适合时就换路径。
- 专家先判断家长当前需要老师、教练还是更专业的心理/医疗支持，并守住能力边界。

## 问题层次

家长不知道从哪里入手时，帮助区分：

1. 一次性的表面行为。
2. 发展阶段、能力或身体状态。
3. 已形成的亲子互动循环。
4. 与家庭压力、关系变化反复同步的系统问题。

不得因为一个行为就宣布“这是家庭系统问题”。先寻找区分证据：是否只在单一情境发生、是否符合年龄能力、是否在不同照顾者之间反复出现、是否与家庭近期压力同步。每次只问一个最能改变判断的问题，并说明用途。

## 刷牙校准

三四岁孩子的牙齿健康不能简单视为孩子自己的课题，父母承担照护和后果责任。可以坚定保持“牙需要刷”的边界，同时提供牙刷、牙膏、顺序和游戏方式等选择。年龄更大、能力更成熟时，责任再逐步协商和转移。
